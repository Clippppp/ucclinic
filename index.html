#이름학교학년 신청 시간대클리닉 종류세부 내용연락처제출시각
📋 구글 시트 연동 방법 (클릭하여 펼치기)
  1. 구글 드라이브에서 새 Google 스프레드시트를 만듭니다.
  2. 상단 메뉴 → 확장 프로그램 → Apps Script를 클릭합니다.
  3. 아래 코드를 붙여넣고 저장(Ctrl+S)합니다:
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['번호','이름','학교','학년','연락처','신청시간대','클리닉종류','세부내용','추가요청사항','제출시각']);
  }
  sheet.appendRow([
    sheet.getLastRow(),
    data.name, data.school, data.grade, data.contact,
    data.timeSlot, data.subject, data.detail, data.note,
    new Date().toLocaleString('ko-KR')
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({result:'success'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService
    .createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}
  1. 상단 메뉴 → 배포 → 새 배포를 클릭합니다.
  2. 유형: 웹 앱 선택 → 액세스: 모든 사용자로 설정합니다.

🔗 학생 공유 링크 만들기 — 무료 배포 방법 (클릭하여 펼치기)

아래 두 가지 방법 중 하나로 HTML 파일을 배포하면 학생들에게 나눌 수 있는 고정 URL이 생깁니다. 두 방법 모두 완전 무료입니다.

✦ 추천 · 가장 쉬운 방법
방법 1 — Netlify Drop (드래그 앤 드롭)
1
app.netlify.com/drop 에 접속합니다.
2
다운로드한 clinic-signup.html 파일을 화면에 드래그 앤 드롭합니다.
3
자동으로 배포되며 https://랜덤.netlify.app 형식의 URL이 생성됩니다.
4
해당 URL을 카카오톡, 문자 등으로 학생들에게 공유합니다. (무료 계정 생성 시 URL 커스텀도 가능)
GitHub Pages
방법 2 — GitHub Pages
1
github.com 에서 무료 계정을 만들고 New Repository를 클릭합니다.
2
파일명을 index.html 로 바꾼 뒤 업로드합니다.
3
저장소 설정 → Pages → Source: main branch 로 설정합니다.
4
https://사용자명.github.io/저장소명 주소로 접속 가능해집니다.

💡 내용 수정 후에는 파일을 다시 업로드(재배포)하면 반영됩니다.
